# StockR

A Tinder-inspired stock discovery platform built with Next.js 14, React, and Tailwind CSS.

## Features

- **Swipe Deck**: Discover stocks through an intuitive swipe interface
- **AI Stock Chats**: Get personalized insights for saved stocks
- **Leaderboard**: Track trending favorites and top StockScore™ ratings
- **Proprietary Scoring**: StockScore™ algorithm evaluates stocks across multiple factors

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- react-swipeable

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

```
src/
├── app/              # Next.js app router pages
├── components/      # React components
├── data/           # Mock stock data
├── lib/            # Utilities and store
└── types/          # TypeScript type definitions
```
